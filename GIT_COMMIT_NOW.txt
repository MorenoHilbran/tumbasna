MASALAH:
Git index.lock file sedang di-lock oleh proses lain.
Ini biasanya terjadi karena:
  • Ada aplikasi Git GUI yang masih buka (GitHub Desktop, GitKraken, dll)
  • Ada proses git lain yang masih berjalan
  • Previous git command yang interrupted

══════════════════════════════════════════════════════════════════

🔧 SOLUSI - IKUTI LANGKAH INI:

STEP 1: Tutup Semua Aplikasi Git
  → GitHub Desktop
  → GitKraken
  → Git GUI
  → VS Code (jika ada Git extension)
  → Sourcetree
  → Any other Git tools

STEP 2: Buka Command Prompt as Administrator
  → Tekan Win + X
  → Pilih "Command Prompt (Admin)" atau "Terminal (Admin)"

STEP 3: Jalankan Command Ini Satu Per Satu:

cd "C:\LIST PROJECT\tumbasna"

REM Kill any running git processes
taskkill /F /IM git.exe /T 2>nul

REM Remove lock file
del /F .git\index.lock 2>nul

REM Verify lock is gone
dir .git\index.lock 2>nul

REM Add all changes
git add .

REM Check status
git status

REM Commit with detailed message

🚨 JIKA MASIH ERROR:

1. Restart Explorer:
   taskkill /F /IM explorer.exe
   start explorer.exe

2. Check running processes:
   tasklist | findstr git

3. Kill all git processes:
   taskkill /F /IM git.exe /T

4. Try again from STEP 3
